# Annual Store Sales Analysis Dashboard (Excel Project)

## 📌 Project Overview

This project is an **interactive Excel Sales Dashboard** created to analyze annual store performance using data visualization and business intelligence techniques.

The dashboard provides insights into:

* Monthly sales performance
* Order trends
* Gender-wise sales distribution
* Order status analysis
* State-wise top sales performance
* Age vs Gender purchasing behavior
* Sales channel contribution

This project demonstrates how **Microsoft Excel** can be used as a powerful data analytics and dashboard-building tool.

## 📈 Key Insights

* Women contribute significantly more to total sales compared to men.
* Delivered orders account for the highest order completion rate.
* Maharashtra is the top-performing state in terms of sales.
* Amazon and Myntra are major contributing sales channels.
* Adult customers generate the highest order volume.
